# site_Exercicio_bandeiras
Exercício feito durante a gradução para treinar o JavaScript
📌 Sobre o Projeto
Durante a disciplina de Programação Web Responsiva, desenvolvi este projeto com o objetivo de praticar e consolidar meus conhecimentos em JavaScript, juntamente com HTML e CSS.
O site consiste em uma interface interativa que apresenta bandeiras de 7 países. Ao clicar em cada bandeira, são exibidas informações relevantes sobre o país selecionado.

🌍 Funcionalidades
Exibição de bandeiras de diferentes países
Interação via clique do usuário
Apresentação dinâmica de informações, como:
Nome do país
Continente
Capital
População
Língua oficial
Forma de governo

🛠️ Tecnologias Utilizadas
HTML → Estrutura da página
CSS → Estilização e layout
JavaScript → Interatividade e manipulação de dados

🎯 Objetivo
Este projeto teve como principal objetivo exercitar a manipulação de eventos em JavaScript e a interatividade em páginas web, proporcionando uma experiência simples e informativa ao usuário.
